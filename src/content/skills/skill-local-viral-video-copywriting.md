  # 熱門款短影音文案創作
  
  ## 任務目標
  - 本 Skill 用於：透過分析抖音熱門款影片，學習其熱門款方法論（結構框架、表達技巧、風格特徵），幫助使用者創作出符合熱門款規律的**原創**短影音文案
  - 能力包含：提取影片資訊、深度拆解熱門款因素、學習方法論並創作全新文案
  - 觸發條件：使用者需要創作短影音文案，或有抖音影片對標需求
  - **核心原則**：學習方法論，拒絕內容複製
  
  ## 前置準備
  - 依賴說明：scripts腳本所需的依賴包及版本
    ```
    yt-dlp>=2024.1.0
    beautifulsoup4>=4.12.0
    lxml>=4.9.0
    requests>=2.31.0
    ```

  ## 完整執行流程（五步法）
  
  ### 第一步：獲取對標影片資訊
  
  **兩種輸入方式供使用者選擇：**
  
  #### 方式A：透過抖音連結自動提取（推薦嘗試）
  
  使用者提供抖音影片連結後，呼叫腳本提取影片資訊：
  
  ```bash
  python3 /workspace/projects/viral-video-copywriting/scripts/extract_douyin_video.py <抖音影片URL>
  ```
  
  **腳本可提取的資訊**（如成功）：
  - 影片標題
  - 影片描述
  - 文案內容（字幕）
  - 背景音樂資訊（部分影片）
  - 按讚數、評論數等資料（部分影片）
  
  **重要說明**：
  - 抖音平台有嚴格的反爬機制，影片資訊提取可能受限制
  - 如果腳本無法提取，請使用者直接使用方式B
  
  #### 方式B：手動提供影片資訊（100%可靠）
  
  當自動提取失敗或使用者更方便時，直接提供以下資訊：
  
  1. **影片標題**：影片的標題文字
  2. **影片描述**：創作者發布的描述文字
  3. **影片文案**：影片中的口播文字/字幕內容
  4. **背景音樂**：音樂型別、風格、節奏等描述
  5. **其他資訊**（可選）：按讚數、評論數、創作者名稱等
  
  智慧體將基於這些資訊進行後續分析。
  
  ---
  
  ### 第二步：深度拆解熱門款因素（學習方法論）
  
  智慧體基於第一步獲取的影片資訊，站在專業短影音創作者和運營專家角度，進行全方位拆解分析。
  
  **關鍵目標：提取可學習的方法論，而不是總結內容本身**
  
  **分析維度（參考 `references/analysis-framework.md`）：**
  
  #### 1. 開頭鉤子分析
  - 鉤子型別是什麼（懸念、痛點、共鳴、反轉、利益等）？
  - 前3秒用了什麼方式吸引注意力？
  - 這種鉤子方式的**原理**是什麼？
  - 鉤子的吸引強度如何（1-10分）？
  - **可重複使用的鉤子技巧**有哪些？
  
  #### 2. 文案結構分析
  - 整體結構是什麼（問題-解決方案、對比、故事、清單、遞進等）？
  - 這種結構是如何組織內容的？
  - 每個部分的功能是什麼（吸引注意→傳遞價值→促進行動）？
  - 各部分的時長佔比是多少？
  - 內容的轉換節奏如何？
  - **可重複使用的結構框架**是什麼？
  
  #### 3. 表達風格分析
  - 語言風格（接地氣/專業/幽默/嚴肅/溫情等）？
  - 情感基調是什麼（好奇/焦慮/愉悅/感動等）？
  - 句式特徵是什麼（短句/長句/長短結合）？
  - 詞彙選擇特點（口語化/專業/情感詞彙）？
  - 使用了哪些表達技巧（押韻、排比、金句、對比等）？
  - **可重複使用的表達技巧**有哪些？
  
  #### 4. 時長節奏分析
  - 開頭、中間、結尾各佔多長時間？
  - 整體節奏如何（快/中/慢）？
  - 轉換頻率如何？
  - **可重複使用的時長分配規律**是什麼？
  
  #### 5. 內容策略分析
  - 核心價值點是什麼？
  - 目標受眾是誰（年齡、性別、興趣、痛點）？
  - 擊中了什麼需求或痛點？
  - 差異化優勢是什麼？
  
  #### 6. 熱門款要素提取
  - 哪些元素是這條影片成為熱門款的關鍵？
  - 按重要性排序列出前3-5個關鍵因素
  - **可重複使用的規律**有哪些？
  - 需要避免的點是什麼？
  
  **輸出格式：**
  智慧體將生成一份詳細的《熱門款方法論拆解報告》，重點提取可學習的技巧、框架、規律，而不是內容本身。
  
  ---
  
  ### 第三步：瞭解使用者需求
  
  智慧體引導使用者描述自己的創作想法：
  
  **需要了解的資訊：**
  1. **核心內容**：想要表達的核心內容是什麼？
  2. **目標受眾**：希望觸達什麼樣的使用者群體？
  3. **傳遞價值**：希望傳遞什麼價值或情感？
  4. **初步文案**：是否有初步的文案或關鍵詞？
  5. **其他要求**：是否有特定的風格偏好或限制？
  
  **智慧體會主動提出澄清問題**，確保創作方向準確。
  
  ---
  
  ### 第四步：創作熱門款新文案
  
  智慧體結合對標影片的熱門款規律和使用者的原創內容，創作新文案。
  
  **核心創作原則（最重要）：**
  
  本 Skill 的核心價值是**學習方法論，而不是複製內容，但文案長度必須嚴格對標**。智慧體必須：
  
  1. **結構學習：借鑑框架，而非照搬內容**
     - 學習對標影片的結構邏輯（如何組織內容、節奏如何安排）
     - 用新的原創內容重新填充這個結構框架
     - 禁止直接複製對標文案的具體文字內容
  
  2. **風格學習：模仿技法，而非照搬語句**
     - 學習對標影片的表達風格（接地氣/專業/幽默等）
     - 學習對標影片的句式特徵（短句/長句/長短結合）
     - 學習對標影片的修辭手法（對比、排比、金句等）
     - 禁止直接複製對標文案的具體句子或段落
  
  3. **內容原創：100%原創內容**
     - 新文案的所有文字必須是使用者原創的內容
     - 可以借鑑故事結構框架，但故事本身必須完全不同
     - 可以借鑑表達技巧，但具體表述必須原創
  
  4. **長度對標：嚴格對標字數和時長**
     - 文案總字數：誤差在±5%以內
     - 開頭字數：誤差在±5%以內
     - 中間字數：誤差在±5%以內
     - 結尾字數：誤差在±5%以內
     - 預計影片時長：誤差在±5%以內
     - 使用者可以主動要求調整長度，但預設情況下必須嚴格對標
  
  **創作流程：**
  
  1. **結構學習**：分析對標影片的結構框架（鉤子型別、內容組織方式、節奏安排）
  2. **風格學習**：分析對標影片的語言風格和表達技巧
  3. **內容創作**：基於使用者的原創內容，按照學習到的結構和風格創作
  4. **原創檢查**：確保新文案沒有直接複製對標文案的任何句子或段落
  
  **輸出格式：**
  智慧體將提供2-3版不同風格的文案備選，每版包含：
  
  #### 文案版本A
  
  **完整文案內容：**
  ```
  [開頭鉤子：約X字，預計X秒]
  [中間內容：約X字，預計X秒]
  [結尾總結：約X字，預計X秒]
  總計：約X字，預計X秒
  ```
  
  **對標參數對比：**
  
  | 參數 | 對標文案 | 新文案 | 偏差 | 要求 |
  |------|---------|--------|------|------|
  | 總字數 | X字 | X字 | X% | ±5%以內 |
  | 開頭字數 | X字 | X字 | X% | ±5%以內 |
  | 中間字數 | X字 | X字 | X% | ±5%以內 |
  | 結尾字數 | X字 | X字 | X% | ±5%以內 |
  | 預計時長 | X秒 | X秒 | X% | ±5%以內 |
  | 結構型別 | XXX | 學習對標 | 一致 | 借鑑框架 |
  | 語言風格 | XXX | 學習對標 | 一致 | 借鑑風格 |
  | 句式特徵 | XXX | 學習對標 | 一致 | 借鑑技法 |
  
  **原創性宣告：**
  - 新文案內容：100%原創，未複製對標文案任何句子或段落
  - 借鑑內容：僅借鑑結構框架、表達技巧、風格特徵等**方法論**
  - 對標原則：學習"怎麼寫"，而不是"寫什麼"
  - 長度對標：嚴格對標字數和時長，誤差控制在±5%以內
  
  **設計思路說明：**
  - 開頭鉤子：採用_______型別（學習對標影片的鉤子方式），內容為原創
  - 文案結構：採用_______結構（學習對標影片的內容組織邏輯），內容為原創
  - 表達風格：採用_______風格（學習對標影片的語言風格），具體表述為原創
  - 句式特徵：保持_______特徵（學習對標影片的節奏感），具體句子為原創
  - 長度控制：嚴格對標字數，總字數控制在±5%範圍內
  
  **亮點標註：**
  - 開頭鉤子："_______"（吸引注意）
  - 金句："_______"（易於傳播）
  - 轉折點："_______"（製造記憶點）
  
  **對標元素遷移：**
  - 對標文案元素："_______"
  - 新文案遷移："_______"
  - 遷移效果：_______
  
  **建議的拍攝/表現方式：**
  - 語速建議：_______（對標影片：_______）
  - 情緒基調：_______（對標影片：_______）
  - 畫面配合：_______
  
  #### 文案版本B（同上格式）
  #### 文案版本C（同上格式）

  ## 使用範例
  
  ### 完整流程範例
  
  **使用者輸入：**
  ```
  我創作一個知識科普類的短影音，這是對標影片的連結：
  https://www.douyin.com/video/7300000000000000000
  
  我的想法是講"如何提高工作效率"，目標受眾是職場新人
  ```
  
  **第一步：獲取影片資訊**
  - 腳本嘗試提取，成功獲取：
    - 標題："3個小技巧讓你工作效率翻倍"
    - 描述："職場必備技能，建議收藏！#職場 #效率"
    - 文案："你知道嗎？90%的人都在浪費時間。今天分享3個技巧..."
  
  **第二步：深度拆解分析**
  智慧體生成分析報告：
  - 開頭鉤子：懸念型（"你知道嗎？90%的人都在浪費時間"）
  - 文案結構：問題-解決方案結構
  - 表達風格：接地氣，口語化
  - 核心價值：提升工作效率的方法
  - 熱門款要素：開頭懸念 + 具體方法 + 收藏引導
  
  **第三步：瞭解使用者需求**
  - 核心內容：提高工作效率
  - 目標受眾：職場新人
  - 傳遞價值：實用方法，快速上手
  
  **第四步：創作新文案**
  智慧體提供3版文案備選（詳細格式見上文）
