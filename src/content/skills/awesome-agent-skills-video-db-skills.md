## 這個 Skill 在做什麼

Realtime 與 batch video 工作流: capture screen/audio, ingest URLs/YouTube/RTSP, transcribe, index, search, generate subtitles, edit timelines, 與 stream HLS output。這筆資料來自 `awesome-agent-skills` 的索引清單，原始專案是 `video-db/skills`。

## 為什麼值得收錄

它屬於 Specialized Domains 相關的 Agent Skill，可作為尋找、安裝或評估同類 Skill 的入口。因為來源 repo 本身沒有把完整 Skill 檔案放在本地，所以此頁保留 catalog 資訊與原始連結，避免把索引項誤認為完整 `SKILL.md`。
