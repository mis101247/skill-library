  # 小紅書美妝內容創作技能
  
  ## 任務目標
  - 本技能用於：為美妝護膚產品建立小紅書平台的高品質筆記內容和配圖提示詞
  - 能力包含：
    1. 筆記生成：根據商品資訊、賣點、目標使用者生成符合小紅書風格的內容筆記
    2. 筆記最佳化：最佳化已有筆記的標題、正文、標籤，提升吸引力和傳播力
    3. 文生圖描述：為美妝內容生成專業的AI繪畫提示詞
  - 觸發條件：使用者提出小紅書筆記創作需求、筆記最佳化需求，或需要為美妝內容生成配圖提示詞時
  
  ## 前置準備
  - 無需特殊依賴包
  - 無需非標準檔案/資料夾
  
  ## 操作步驟
  
  ### 標準流程：筆記生成
  
  1. **收集商品資訊**
     - 詢問使用者提供：產品名稱、品牌、價格、核心成分、主要功效、適合膚質、使用場景
     - 確定目標使用者畫像：年齡段、膚質、消費能力、需求痛點
  
  2. **選擇筆記型別**
     - 根據[references/note_templates.md](references/note_templates.md)選擇合適的模板型別：
       - 產品測評型：適合產品詳細測評
       - 護膚乾貨型：適合知識分享
       - 成分科普型：適合成分解析
       - 痛點解決型：適合解決具體問題
       - 對比測評型：適合多產品對比
       - 使用教程型：適合教程分享
       - 好物合集型：適合多產品推薦
       - 經驗分享型：適合心得分享
  
  3. **創作筆記內容**
     - **標題創作**：使用emoji、突出賣點、製造緊迫感（參考 note_templates.md 中的標題最佳化技巧）
     - **開頭鉤子**：製造共鳴、突出效果、給予承諾
     - **正文結構**：
       - 產品資訊（名稱、品牌、價格、規格）
       - 使用感受（質地、味道、上臉感、效果）
       - 核心賣點（功效、成分、適合人群）
       - 使用方法（步驟、注意事項）
       - 避雷提醒（不適合人群、使用禁忌）
       - 總結推薦（綜合評價、價效比）
     - **標籤組合**：根據[references/tag_library.md](references/tag_library.md)選擇10-15個核心標籤
     - **語氣風格**：親切自然、使用"姐妹們"、"家人們"等稱呼、適當使用感嘆詞
  
  4. **最佳化筆記品質**
     - 檢查是否符合小紅書風格：emoji使用、段落分隔、重點突出
     - 確保內容真實可信：避免過度誇大、提供真實感受
     - 突出產品賣點：核心功效、獨特優勢、價效比
     - 增加互動感：使用問句、引導評論、鼓勵分享
  
  ### 標準流程：筆記最佳化
  
  1. **分析原始筆記**
     - 識別問題點：標題吸引力不足、正文結構混亂、標籤不精準
     - 評估改進空間：可最佳化的方向（標題、內容、標籤）
  
  2. **標題最佳化**
     - 新增emoji吸引眼球
     - 突出賣點（價效比、效果、適合人群）
     - 製造緊迫感（必入、閉眼入、無限回購）
     - 使用感嘆詞（絕絕子、太香了、好用到哭）
  
  3. **正文最佳化**
     - 最佳化開頭：製造共鳴、突出效果、給予承諾
     - 最佳化結構：使用emoji分段、數字標記、重點加粗
     - 最佳化內容：增加細節描述、提供真實感受、突出賣點
     - 最佳化結尾：總結推薦、引導互動
  
  4. **標籤最佳化**
     - 根據[references/tag_library.md](references/tag_library.md)重新選擇標籤
     - 確保標籤與內容匹配
     - 控制標籤數量（10-15個）
     - 結合熱門標籤
  
  ### 標準流程：文生圖描述創作
  
  1. **瞭解配圖需求**
     - 確定圖片用途：產品展示、人物肖像、美妝場景、氛圍感圖
     - 明確風格要求：溫柔清新、精緻奢華、青春活力、溫柔治癒
  
  2. **創作提示詞**
     - 參考[references/image_prompt_guide.md](references/image_prompt_guide.md)中的提示詞模板
     - 按照公式構建提示詞：主體 + 描述 + 場景 + 風格 + 細節 + 構圖
     - 選擇合適的描述詞庫：
       - 皮膚描述：光澤皮膚、清透底妝、完美皮膚
       - 妝容描述：精緻妝容、大地色眼影、豆沙色口紅
       - 光線描述：自然光、柔和光線、晨光
       - 風格描述：溫柔清新、精緻奢華、青春活力
       - 構圖描述：半身像、居中構圖、背景虛化
  
  3. **最佳化提示詞**
     - 突出重點：把重要詞放在前面
     - 細節豐富：描述質感、細節、光線
     - 風格明確：指定藝術風格、色調、氛圍感
     - 場景具體：說明環境、元素、氛圍
  
  ## 資源索引
  
  - **筆記模板庫**：見[references/note_templates.md](references/note_templates.md)（用途：提供8種型別的完整筆記模板）
  - **標籤庫**：見[references/tag_library.md](references/tag_library.md)（用途：提供分類標籤和組合建議）
  - **美妝趨勢參考**：見[references/makeup_trends.md](references/makeup_trends.md)（用途：瞭解當前熱門美妝趨勢、成分、功效）
  - **文生圖提示詞指南**：見[references/image_prompt_guide.md](references/image_prompt_guide.md)（用途：創作專業的AI繪畫提示詞）
  
  ## 注意事項
  
  - 內容真實性：避免過度誇大產品效果，提供真實使用感受
  - 小紅書風格：使用emoji、親切語氣、符合平台調性
  - 標籤精準度：根據筆記內容選擇匹配的標籤，避免無關標籤
  - 圖文一致：文生圖描述要與筆記內容、產品調性保持一致
  - 避免敏感詞：不使用絕對化用語、不涉及醫療功效、不誇大宣傳
  - 版權意識：描述中避免直接引用品牌Logo、商標
  
  ## 使用範例
  
  ### 範例1：產品測評型筆記生成
  - **需求**：為某款美白精華創作小紅書筆記
  - **執行方式**：智慧體自然語言創作
  - **關鍵要點**：
    - 收集產品資訊（價格、成分、功效）
    - 選擇產品測評型模板
    - 創作標題："✨挖到寶了！這個美白精華真的絕絕子！相見恨晚😭"
    - 撰寫正文（產品資訊、使用感受、避雷提醒、總結）
    - 選擇標籤：#護膚推薦 #美白 #煙醯胺 #學生黨 #平價好物
    - 最佳化排版（emoji分段、重點突出）
  
  ### 範例2：筆記最佳化
  - **需求**：最佳化現有護膚筆記，提升吸引力
  - **執行方式**：智慧體自然語言最佳化
  - **關鍵要點**：
    - 分析原始筆記問題點
    - 最佳化標題：新增emoji、突出賣點
    - 最佳化正文：改進結構、增加細節
    - 最佳化標籤：重新選擇精準標籤
    - 檢查排版和風格
  
  ### 範例3：文生圖描述創作
  - **需求**：為護膚品創作產品展示圖提示詞
  - **執行方式**：智慧體自然語言創作
  - **關鍵要點**：
    - 確定圖片用途（產品展示）
    - 參考產品展示提示詞模板
    - 構建完整提示詞（主體+描述+場景+風格+細節+構圖）
    - 最佳化描述（突出質感、光線、風格）
    - 輸出專業提示詞供AI繪畫使用
  
  ## 品質標準
  
  - 筆記內容：
    - 標題吸引眼球，有emoji和賣點
    - 正文結構清晰，邏輯流暢
    - 內容真實可信，無過度誇大
    - 標籤精準匹配，數量適中
  - 文生圖描述：
    - 主體明確，描述詳細
    - 風格一致，氛圍感強
    - 細節豐富，畫面感強
    - 符合產品調性和平台風格
