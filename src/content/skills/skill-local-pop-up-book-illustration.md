  # 3D紙藝彈出書插畫生成
  
  ## 任務目標
  - 本 Skill 用於：生成具有3D立體紙藝效果的彈出書風格插畫
  - 能力包含：結合紙質紋理與數字科技元素，創造溫暖而富有科技感的視覺作品
  - 觸發條件：使用者需要生成具有紙藝質感、立體層次、特定色調（薄荷綠/天藍/珊瑚橙/奶油黃）的插畫時
  
  ## 核心設計要素
  
  ### 色彩方案
  - 主色調：薄荷綠（清新輕盈）
  - 輔助色：天藍色（科技感）、珊瑚橙（活力）、奶油黃（溫暖）
  - 整體氛圍：明亮溫暖，避免深沉或冷峻色調
  
  ### 視覺特徵
  - 紙質紋理質感：紙張表面觸感、摺痕、紋理細節
  - 立體層次：多層彈出結構，前後景深明顯
  - 書本場景：開啟的書本，書頁邊緣可見，營造真實翻書效果
  - 數字科技元素：霓虹光效、科技圖示、抽象資料流，與紙質風格融合
  
  ### 效果細節
  - 柔和霓虹光效：點綴式光效，不破壞紙質質感
  - 溫暖科技光照：暖色調光照配合科技元素
  - 立體陰影：增強層次感和真實感
  - 細節豐富：包含精美的裝飾元素和關鍵資訊展示
  
  ### 風格定位
  - 整體風格：圓潤可愛但專業精緻
  - 視覺遞進：從簡單到複雜的視覺層次
  - 質感結合：紙質手工質感與數字科技感的平衡
  
  ## 操作步驟
  
  ### 步驟1：理解場景需求
  - 確認主題內容：科技、教育、品牌、產品等
  - 明確關鍵資訊：需要突出的文字、圖示、元素
  - 確定複雜程度：單層彈出vs多層立體結構
  
  ### 步驟2：構建影像生成提示詞
  根據核心設計要素，構建包含以下元素的提示詞：
  
  **基礎結構：**
  ```
  3D紙藝彈出書風格插畫，立體層次感，開啟的書本場景，書頁邊緣清晰可見
  ```
  
  **色彩新增：**
  ```
  明亮溫暖色調，薄荷綠、天藍色、珊瑚橙、奶油黃配色
  ```
  
  **質感描述：**
  ```
  紙質紋理質感，摺痕細節，手工製作感，紙張表面觸感
  ```
  
  **立體效果：**
  ```
  多層彈出結構，前後景深，立體陰影，真實立體感
  ```
  
  **科技元素（如需要）：**
  ```
  柔和霓虹光效點綴，數字科技元素，科技圖示，資料流線條，溫暖科技光照
  ```
  
  **風格特徵：**
  ```
  圓潤可愛但專業精緻，細節豐富，精美裝飾
  ```
  
  ### 步驟3：生成並驗證
  - 生成影像後檢查：
    - 立體層次是否明顯
    - 色調是否符合明亮溫暖要求
    - 紙質質感是否真實
    - 科技元素是否與紙質風格融合自然
    - 細節是否豐富但不雜亂
  
  ## 使用範例
  
  ### 範例1：科技主題彈出書
  **功能說明**：展示AI技術與紙質工藝的結合
  
  **提示詞構建：**
  ```
  3D紙藝彈出書風格插畫，開啟的書本中央彈出AI大腦圖案，周圍是神經網路線條，紙質紋理質感，立體層次明顯，薄荷綠和天藍色為主色調，柔和霓虹光效點綴，溫暖科技光照，書頁邊緣可見，圓潤可愛但專業精緻
  ```
  
  ### 範例2：教育主題彈出書
  **功能說明**：展示知識探索場景
  
  **提示詞構建：**
  ```
  3D紙藝彈出書風格插畫，開啟的書本彈出多層知識樹，樹上有書本、燈泡、齒輪等元素，珊瑚橙和奶油黃為主色調，紙質摺痕細節豐富，立體陰影增強層次感，書頁邊緣清晰，從簡單到複雜的視覺遞進，溫暖明亮氛圍
  ```
  
  ### 範例3：品牌產品展示彈出書
  **功能說明**：展示產品或品牌故事
  
  **提示詞構建：**
  ```
  3D紙藝彈出書風格插畫，開啟的書本中央彈出品牌產品，周圍環繞品牌關鍵詞和裝飾元素，薄荷綠和天藍色配合珊瑚橙點綴，紙質紋理與數字科技元素結合，柔和霓虹光效，立體多層結構，書頁邊緣真實可見，專業精緻且溫暖可愛
  ```
  
  ## 注意事項
  - 保持紙質質感的主導地位，科技元素為輔助點綴
  - 確保立體層次清晰，避免扁平化效果
  - 色調始終保持明亮溫暖，避免冷峻或暗沉
  - 細節豐富但要有視覺重心，避免過於雜亂
  - 書頁邊緣和翻書效果是關鍵識別特徵，需清晰呈現
  
  ## 適用場景
  - 科技產品宣傳：展示AI、雲服務、資料等產品特性
  - 教育內容呈現：知識圖譜、學習路徑、課程大綱
  - 品牌故事講述：品牌理念、發展歷程、核心價值
  - 資訊圖表設計：資料視覺化、流程說明、層級關係
  - 活動視覺物料：海報、邀請函、宣傳冊封面
