## 這個 Skill 在做什麼

Document processing 搭配 Nutrient DWS API: convert (PDF/DOCX/XLSX/PPTX/HTML/images), extract text/tables, OCR (20+ languages), redact PII (pattern + AI), watermark, digital signatures, form filling. [MCP server](https://www.npmjs.com/package/@nutrient-sdk/dws-mcp-server) also available。這筆資料來自 `awesome-agent-skills` 的索引清單，原始專案是 `PSPDFKit-labs/nutrient-agent-skill`。
